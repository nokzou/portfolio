Dodge the Cars


Class: CSE
Date: Sep 8, 2017
"Dodge the Cars" was my first project, and was a game that I made using MIT's Scratch website. My main goal was to simply make a replayable game with simple mechanics, but a great difficulty. I came up with these ideas based on the average popular smartphone game like Flappy Bird or Crossy Road, as they have constant, repeated mechanics that progressively get harder over time. As my first start, I tried to focus on the foundational aspect rather than trying to mesh out everything that I had imagined when designing the game. I tried to develop good programming habits using my experience in creating this game rather than take a huge leap into programming.

Hangman


Class: CSE
Date: Feb 1, 2018
Our “Hangman” project was made with Python. I created it in collaboration with Sahithi Samsani, like I did with the previous project. My main job in this project was to develop a looping function that would essentially play Hangman with the user by generating a random word, asking them to input letters, and showing the parts of the word that they had guessed correctly. I also made ASCII art to show on each turn. Through programming this project, I, again, improved my understanding of Python and worked on my ability to collaboratively program with another person at the same time on the same file.
